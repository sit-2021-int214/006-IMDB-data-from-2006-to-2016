## Part 2: Learning function from Tidyverse

- Function `str_detect(x, pattern)` from package [stringr](https://stringr.tidyverse.org/reference/str_detect.html)). It using for detect the presence or 
- absence of a pattern in a string

```

#> [1] TRUE TRUE TRUE TRUE
```
